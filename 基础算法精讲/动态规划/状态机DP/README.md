## 恰好 k 次/至少 k 次

### 恰好

`f[0][1][0]=0, 其余=-inf`

**注意** 前面塞了个状态，`f[0][1]`才是恰好完成 `0` 次的状态

### 至少

`f[i][-1][·]` 等价于 `f[i][0][·]`

所以每个 `f[i]` 的最前面不需要插入状态

「至少0次」等价于「可以无限次交易」

所以 `f[i][0][·]` 就是无限次交易下的最大利润，转移方程也一样

`f[0][O][0] =0, 其余=-inf`

`f[i+1][0][0] =max(f[i][0][0],f[i][0][1] + prices[i])`

`f[i+1][o][1] =max(fii][o][1],fi][o][o] - prices[i])`
